Infludot Web Platform

Author: Abhinav
Developed during internship at Infludot

---

Overview

This project is a web platform developed using React for the frontend and FastAPI for the backend.
It includes a user-facing website, a RAG-based chatbot, and an admin panel for managing brands and influencers.

Project Overview Video:
https://github.com/user-attachments/assets/43401d4b-58fd-4e05-81e1-89acf8116028

---

Features

- User-facing web platform
- RAG-based chatbot
- Admin panel for managing brands and influencers
- REST API backend built with FastAPI
- React-based frontend
- JWT-based authentication
- Modular project structure

---

Tech Stack

Frontend: React
Backend: FastAPI (Python)
Database: PostgreSQL
