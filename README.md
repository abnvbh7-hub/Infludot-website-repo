🚀 Infludot Web Platform

Author: Abhinav
Developed during internship at Infludot

---

📌 Overview

A web platform built with React (frontend) and FastAPI (backend) featuring a RAG-based chatbot and an admin panel for managing brands and influencers.

🎥 Project Overview Video


https://github.com/user-attachments/assets/4e32863f-f520-434a-9f9f-8d49579e1fbc

---

✨ Key Features

- 🌐 User-facing web platform
- 🤖 RAG-based chatbot
- 🛠️ Admin panel for brands & influencers
- 🔐 JWT-based authentication
- ⚡ FastAPI REST backend
- ⚛️ React frontend
- 🧩 Modular architecture

---

🧰 Tech Stack

- Frontend: React ⚛️
- Backend: FastAPI (Python) 🐍
- Database: PostgreSQL 🐘
