Infludot Web Platform

This project was developed during my internship at Infludot.
It is a web platform built using React for the frontend and FastAPI for the backend.

Features:

- User-facing website
- RAG based Chatbot
- Separate admin panel for management
- REST API backend using FastAPI
- Frontend built with React
- Modular project structure
- Secured JWT authentication

Note:
-The current version is connected to my local/development database configuration.
-Database credentials and configuration may need to be updated for the production environment.

Tech Stack:
Frontend: React
Backend: FastAPI (Python)
Database: Postgres
